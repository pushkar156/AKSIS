# AKSIS - Industrial Automation Systems Integrator

[![License: Proprietary](https://img.shields.io/badge/License-Proprietary-red.svg)](LICENSE)

**AKSIS** is a professional, high-performance website designed for **AKSIS Industrial Technologies**, a multidisciplinary engineering integrator specializing in industrial automation for process manufacturers. This project was developed as a freelance assignment to showcase the company's expertise in processing, manufacturing, material handling, packaging, and high-quality industrial solutions.

---

## 🚀 Project Overview

The AKSIS website serves as a digital storefront and resource hub for industrial automation services. It provides detailed insights into:
- **Keys of Business**: Machinery equipment, consultancy, raw materials, software technology, and more.
- **Expertise**: Comprehensive solutions for process systems, machine vision, robotics, and industrial automation.
- **Product Highlights**: Showcasing diverse industrial products and services.
- **Service Integration**: Seamlessly connecting clients with specialists via advanced contact and inquiry forms.

---

## ✨ Features

- **Modern Responsive Design**: Fully optimized for desktops, tablets, and mobile devices using flexible CSS layouts.
- **Dynamic Navigation**: Multi-level dropdown menus for easy exploration of "Keys," "Expertise," and "About" sections.
- **Interactive UI Components**: Custom-built image sliders, scrollers, and scroll-reveal animations for an engaging user experience.
- **Secure Handling of Inquiries**: Dedicated PHP-powered form handlers (`contact-handler.php`, `apply_form_handler.php`) with server-side validation and file upload support for resumes and project documents.
- **Advanced Search Integration**: Built-in search functionality (`search.js`) to help users find relevant products and services instantly.
- **Localized Content Maps**: Integrated Google Maps to provide office location details and directions.

---

## 🛠️ Tech Stack

- **Frontend**: 
  - **HTML5**: Semantic structure for SEO and accessibility.
  - **CSS3**: Vanilla CSS with custom animations, grid-based layouts, and responsive media queries.
  - **JavaScript (ES6+)**: Custom scripting for sliders, dynamic menus, and UI interactivity.
- **Backend / Integration**:
  - **PHP**: Server-side processing for complex forms and file uploads.
  - **Google Fonts & FontAwesome**: Professional typography and iconography.

## ⚙️ Local Setup

To run this project locally for testing or development purposes:

1. **Prerequisites**: Ensure you have a local server environment (e.g., **XAMPP**, **WAMP**, or **MAMP**) to execute the PHP form handlers.
2. **Clone the Repo**:
   ```bash
   git clone <repository_url>
   ```
3. **Run Locally**: Place the project folder in your server's root directory (e.g., `htdocs` for XAMPP).
4. **Access via Browser**: Navigate to `http://localhost/AKSIS/index.html`.
